# Digital Farm App (ডিজিটাল খামার)

A digital farming application to keep track of poultry farm records.

## Features
- Dashboard
- Batch Management
- Feed Records
- Medicine Records
- Mortality Records
- Sales Records
- Expenses & Dues

## Tech Stack
- React
- Vite
- Tailwind CSS
- Firebase (Auth & Firestore)
- PWA (Progressive Web App) Support Added
